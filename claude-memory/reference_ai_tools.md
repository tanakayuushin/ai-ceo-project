## 使いどころの原則

- クライアントへのAI提案時 → 業種別おすすめを参照して具体名を出す
- 自社業務の自動化・効率化 → 自社活用リストから選ぶ
- 競合・市場調査 → リサーチ系ツールを使う
- コンテンツ制作 → 動画・音声・画像系を使う
