回答前に必ずWebSearchで最新情報を調査してから伝える。

**Why:** 過去にX APIの無料枠廃止など、情報が古くて誤った案内をしてしまった。ユーザーは常に最新・正確な情報を求めている。

**How to apply:** サービスの価格・仕様・プラン・機能について話すときは、記憶に頼らずWebSearchで現時点の情報を確認してから提示する。「〜だったはずです」「〜と思います」ではなく調査結果を根拠にして答える。
