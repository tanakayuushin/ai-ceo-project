## 主要補助金一覧

### 1. IT導入補助金 2025（経済産業省）

| 項目 | 内容 |
|------|------|
| 目的 | 中小企業・小規模事業者のITツール導入支援 |
| 補助率 | 1/2〜3/4（枠によって異なる） |
| 補助額 | 5万円〜450万円（枠によって異なる） |
| 対象 | ITツール導入費用（ソフトウェア・クラウド・導入支援費） |
| 申請方法 | IT導入支援事業者（登録ベンダー）経由で申請 |
| 注意点 | （社名）が「IT導入支援事業者」として登録することで顧客の申請を代行できる |

**営業での使い方**:
> 「IT導入補助金を使えば、導入費用の最大75%が戻ってきます。
> 私たちが申請のサポートをしますので、手続きの心配は不要です。」

---

### 2. 小規模事業者持続化補助金（商工会議所経由）

| 項目 | 内容 |
|------|------|
| 目的 | 小規模事業者の販路開拓・業務効率化 |
| 補助率 | 2/3 |
| 補助額 | 上限50万円（通常枠）/ 最大200万円（特別枠） |
| 対象 | ウェブサイト制作・広告・設備・ITツール等 |
| 窓口 | 各地商工会議所・商工会 |
| 注意点 | 商工会議所が申請支援を行うため、商工会議所との関係構築で顧客紹介につながる |

**営業での使い方**:
> 「この補助金、商工会議所さんが申請の窓口なんです。
> セミナーに来てくださった方には優先的にご案内します。」

---

### 3. ものづくり補助金（製造業向け）

| 項目 | 内容 |
|------|------|
| 目的 | 革新的な製品・サービス開発、生産プロセス改善 |
| 補助率 | 1/2〜2/3 |
| 補助額 | 上限750万円〜1,250万円（枠による） |
| 対象 | 設備投資・システム開発・外注費等 |
| ターゲット | 製造業・AIシステム導入を伴う場合に活用可 |

---

### 4. 山口県独自支援（要確認・随時更新）

| 制度名 | 概要 | 確認先 |
|--------|------|--------|
| 山口県DX推進補助金 | 県内中小企業のDX化支援（詳細は県HPで随時確認） | 山口県産業振興財団 |
| やまぐち産業振興財団の支援メニュー | 経営診断・専門家派遣・補助金情報提供 | 産業振興財団 |
| 山口県よろず支援拠点 | 無料経営相談・専門家マッチング | 山口県よろず支援拠点 |

> **CEOメモ**: 山口県独自の補助金は年度ごとに変わるため、定期的に県HP・産業振興財団HPを確認すること。水曜のマーケットリサーチで毎週チェックする。

## （社名）として取るべきアクション

1. **IT導入支援事業者への登録を検討** — 登録することで顧客の補助金申請を代行でき、強力な差別化になる
2. **商工会議所との連携強化** — 持続化補助金の申請窓口が商工会議所のため、関係が深まると顧客紹介に直結する
3. **山口県産業振興財団との接点を作る** — 専門家派遣制度などで協力関係を築けると信頼性が上がる
