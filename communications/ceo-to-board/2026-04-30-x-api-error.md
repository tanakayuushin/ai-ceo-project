## 状況

```
HTTP Error 403: Host not in allowlist
```

## 原因

X APIの開発者アプリ設定において、**投稿を許可するサーバーIPのホワイトリスト**が設定されており、このサーバー（Claude Code実行環境）のIPアドレスがその対象外となっているため投稿が拒否されています。

これはX Developer PortalのアプリEssential/Basic設定に関わる制限で、過去の投稿（4/25〜4/29）もおそらく同じ理由で失敗していた可能性があります。

## 確認依頼

1. **X Developer Portal**（developer.twitter.com）でアプリの「App settings」を確認し、Callback URL / Allowlist の設定を見直していただけますか？
2. または、**手動での投稿**（ツイート文をコピーして@AI_chuusyouアカウントから投稿）に切り替えることを取締役にご判断いただきたいです。
3. 今後のX投稿の運用方針（自動 or 手動）についてご指示をお願いします。
