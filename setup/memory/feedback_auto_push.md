成果物やファイルを作成・更新したときは、アレンが自律的にgit add → git commit → git pushまで実行する。

**Why:** ユーザーが毎回プッシュを指示する手間を省くため。ローカル自動pullと組み合わせて、常にリモートと同期された状態を維持する。

**How to apply:** ファイルを新規作成・編集したセッションの終わりに（または作業区切りで）、コミット＆プッシュまで自動実行する。確認は不要。
