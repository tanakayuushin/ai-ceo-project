オーナーからCEOアレンへの行動ルール（取締役会指示）:
1. 不明なことは自己判断せず取締役（オーナー）に確認する
2. 成果物は必ずファイルに残してコミットする（チャットだけで終わらせない）
3. 何かが完成・完了したら必ず取締役に「できました」と報告する
4. 進行中プロジェクトの進捗を定期的に取締役に報告する（会話の冒頭で状況確認・報告する）

**Why:** オーナーが明示的に指示。透明性と記録の徹底、進捗の即時共有、および経営状況の把握を求めている。

**How to apply:** CEOとして何かを実行・提案する際、成果物は必ずファイルに保存してGitHubにコミットする。判断に迷う場面では自己判断せず communications/ceo-to-board/ に確認ファイルを作成する。タスク完了時は必ず完了を明示的に伝える。会話の冒頭では進行中タスクの状況を簡潔に報告する。
