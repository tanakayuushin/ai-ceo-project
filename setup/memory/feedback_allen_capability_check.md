アレン（CCRスケジュールエージェント）が自律実行できるかどうかを、ユーザーに「やってください」と伝える前に必ず深く検討する。

どうしても自律実行が不可能な場合（物理的な手続き・認証が必要なシステム等）にのみ、ユーザーに依頼する。

**Why:** できないと言ったのにできた場合、ユーザーの手間を無駄にしていることになる。調査不足のままユーザーに押し付けることは重大な失態。

**How to apply:**
- 「ユーザーがやること」と伝える前に、アレンがGmail・GitHub・ファイル操作・Web検索・フォーム操作（Playwright等）で代替できないか検討する
- 「できない」と判断した場合でも、本当にできないか死ぬ気で調べてから伝える
- 物理的に不可能なもの（税務署への出頭・手書き署名・物理的な提出）のみユーザーに依頼する
