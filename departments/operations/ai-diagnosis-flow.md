# AI業務診断 — 提供フロー設計

**作成**: CEO アレン  
**作成日**: 2026-04-21  
**用途**: サービス提供の標準手順。担当者が変わっても同じ品質を維持するため。

---

## サービス概要

| 項目 | 内容 |
|------|------|
| サービス名 | AI業務診断 |
| 所要期間 | 約2週間（訪問1日 + レポート作成1週間） |
| 費用 | 5〜10万円（企業規模・業種により変動） |
| 成果物 | AI活用可能性レポート（A4・10〜15ページ） |
| 目的 | 課題の可視化 + 次のコンサル提案への橋渡し |

---

## フロー全体図

```
お問い合わせ・紹介
      ↓
【STEP 1】初回ヒアリング（30〜60分・オンライン可）
      ↓
【STEP 2】診断契約・費用確定
      ↓
【STEP 3】現場訪問・業務ヒアリング（半日）
      ↓
【STEP 4】レポート作成（1週間）
      ↓
【STEP 5】レポート報告会（60〜90分）
      ↓
【STEP 6】AI導入コンサル提案 or クローズ
```

---

## 各STEPの詳細

### STEP 1：初回ヒアリング（30〜60分）

**目的**: 企業の基本情報・課題感を把握し、診断の範囲を合意する

**確認事項**:
- 業種・従業員数・主要業務の流れ
- 現在困っていること（人手不足・時間がかかる作業・ミスが多い業務など）
- AIや ITツールの現在の利用状況
- 診断対象とする業務の範囲（絞り込む）
- 費用・スケジュールの合意

**チェックリスト**:
- [ ] 企業基本情報メモ
- [ ] 課題・ペインポイント記録
- [ ] 診断スコープ合意
- [ ] 次回日程（訪問日）確定

---

### STEP 2：診断契約・費用確定

**目的**: 正式契約、費用・成果物・スケジュールを書面で合意

**書類**:
- 業務委託契約書（簡易版）
- 診断スコープ確認書
- 請求書（前払い or 完了後払い）

**費用の目安**:
| 規模 | 従業員数 | 診断費用 |
|------|---------|---------|
| 小規模 | 〜10名 | 5万円 |
| 中規模 | 11〜50名 | 7〜8万円 |
| やや大きめ | 51名〜 | 10万円 |

---

### STEP 3：現場訪問・業務ヒアリング（半日 / 3〜4時間）

**目的**: 実際の業務フローを目で見て把握する。現場のリアルを掴む。

**訪問の進め方**:

| 時間 | 内容 |
|------|------|
| 0:00〜0:30 | 経営者・管理職との概況ヒアリング |
| 0:30〜2:00 | 現場スタッフへのヒアリング（実際の業務を見せてもらう） |
| 2:00〜3:00 | 業務フロー・使用ツールの詳細確認 |
| 3:00〜3:30 | まとめ・疑問点の解消・次のステップ確認 |

**ヒアリングで聞くこと**:
- 1日・1週間の業務の流れ
- 時間がかかっている作業トップ3
- ミス・やり直しが多い作業
- 紙・Excelで管理しているもの
- 「これがなくなれば楽なのに」と思う作業

**持参するもの**:
- [ ] ヒアリングシート（印刷）
- [ ] 名刺
- [ ] ノートPC（メモ・録音用）
- [ ] 会社案内（置いてくる用）

---

### STEP 4：レポート作成（訪問後1週間）

**レポートの構成**:

```
表紙
1. 診断概要（企業情報・診断スコープ・実施日）
2. 現状の業務フロー整理
3. AI活用可能性マップ
   ├── 即活用可能（効果大・難易度低）
   ├── 中期的に検討（効果大・難易度中）
   └── 将来的に検討（効果中・難易度高）
4. 推奨AIツール・ソリューション案（3〜5案）
5. 導入ロードマップ（6ヶ月・12ヶ月）
6. 概算費用・補助金活用シミュレーション
7. 次のステップ（Emport AIからの提案）
```

---

### STEP 5：レポート報告会（60〜90分）

**目的**: レポートを説明し、次のアクション（コンサル契約）への合意を取る

**進め方**:

| 時間 | 内容 |
|------|------|
| 0:00〜0:10 | 診断のサマリー（一言で言うと） |
| 0:10〜0:40 | レポートの読み合わせ（重要箇所に絞る） |
| 0:40〜0:60 | 質疑応答 |
| 0:60〜0:75 | 次のステップの提案（コンサル・ツール導入） |
| 0:75〜0:90 | 意向確認・クローズ or 検討期間の合意 |

**クロージングトーク**:
```
「今日のレポートで、御社にとって一番インパクトが大きいのは
○○の業務へのAI導入だと私は考えています。

次のステップとして、実際に導入まで伴走する
コンサルティングをご提案したいのですが、
まずご関心をお聞かせいただけますか？」
```

---

### STEP 6：コンサル提案 or クローズ

**コンサル提案に進む場合**:
- 提案書（スコープ・費用・期間）を1週間以内に送付
- 補助金活用シミュレーションも添付

**今回はクローズの場合**:
- 「検討時期が来たらご連絡ください」と関係を切らない
- 3ヶ月後にフォロー連絡をカレンダーに入れる
- 事例コンテンツをメールで送り続けて接点を保つ
