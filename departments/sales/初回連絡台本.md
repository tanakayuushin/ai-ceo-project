# 台本①：商工会議所への初回コンタクト（電話）

**用途**: 商工会議所の担当部署（中小企業支援課・経営支援課など）への初回電話  
**目標**: セミナー提案書を送る約束を取り付ける or 担当者と話せる日程を確保する

---

## 電話前の確認事項

- [ ] 担当部署名・担当者名を事前に調べておく（HP確認）
- [ ] 電話する時間帯：平日10〜11時 or 14〜16時が繋がりやすい
- [ ] 手元にメモ・提案書を用意しておく

---

## 台本（電話）

```
「お忙しいところ恐れ入ります。
わたくし、（社名）と申しますAIコンサルティング会社の者です。

会員企業様向けのAI活用セミナーについてご提案させていただきたく、
ご連絡いたしました。

山口県内の中小企業様を対象に、
業務効率化のためのAI活用入門セミナーを
【無料で】開催させていただく提案でございます。

御所のご負担はございません。
会員様への付加価値として、ご活用いただければと思っております。

ご担当の方にご提案書をお送りしてもよろしいでしょうか？」
```

---

## 相手の反応別 切り返し

**「どんな会社ですか？」と聞かれたら**
```
「山口県を拠点に、地元の中小企業様のAI導入を支援している会社です。
難しそうに見えるAIを、現場で実際に使えるレベルまで落とし込むことを
専門にしております。」
```

**「今は予定がいっぱいで…」と言われたら**
```
「承知いたしました。では、3ヶ月先・半年先のご予定でも構いません。
まず資料だけでもご覧いただけますか？
メールアドレスをお教えいただければ、すぐにお送りいたします。」
```

**「費用はかかりますか？」と聞かれたら**
```
「御所・参加企業様へのご費用は一切いただきません。
私どもにとっては地域への貢献であり、
地元企業様にAIを知っていただくことが目的です。」
```

**「上の者に確認します」と言われたら**
```
「ありがとうございます。では、ご確認いただける方のお名前と、
改めてご連絡してもよい日時を教えていただけますか？」
```

---

## 電話後にやること

- [ ] 対話内容をメモ → `departments/sales/contact-log.md` に記録
- [ ] 提案書をメールで送付（`chamber-proposal.md` を参照）
- [ ] 1週間後にフォロー連絡の日程を設定
