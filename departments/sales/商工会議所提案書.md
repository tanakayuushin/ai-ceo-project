## ご提案の概要

このたびは、貴所会員企業様を対象とした **AI活用入門セミナーの無料開催** をご提案申し上げます。

近年、AI技術は大企業だけでなく中小企業においても業務効率化・人手不足解消の有力な手段として注目されています。しかしながら、地方の中小企業においては「何から始めればよいかわからない」「コストが心配」「使いこなせる人材がいない」といったお声が多く、導入が進んでいないのが現状です。

本セミナーでは、こうした不安を解消し、会員企業様がAIを身近に感じ、最初の一歩を踏み出せるきっかけを提供いたします。

## セミナー概要

| 項目 | 内容 |
|------|------|
| タイトル | 「中小企業のためのAI活用入門 — 明日から使える業務効率化のヒント」 |
| 対象 | 会員企業の経営者・管理職の方 |
| 時間 | 90分（講義60分 + 質疑応答30分） |
| 定員 | 20〜30名 |
| 参加費 | 無料 |
| 会場 | 貴所のご指定会場（またはオンライン対応可） |
| 開催費用 | 貴所のご負担なし |

## セミナー内容（アジェンダ）

**第1部：なぜ今、中小企業にAIが必要か（15分）**
- 人手不足・生産性向上の現状
- AIで実際に変わった地方中小企業の事例紹介

**第2部：AIで変えられる業務とは（25分）**
- 問い合わせ対応・書類作成・データ整理など身近な業務への活用
- 導入コストの現実（思ったより安い）
- 補助金・支援制度の活用方法

**第3部：どこから始めればいいか（20分）**
- 失敗しないAI導入の進め方
- AI業務診断サービスのご紹介
- スモールスタートの実践ステップ

**質疑応答（30分）**

## 主催者について

**Emport AI**は、山口県を拠点とする企業向けAI活用支援会社です。地元の中小企業が「使いこなせるAI」を実現できるよう、現場に寄り添った支援を行っています。

- 地元密着：山口県内企業への現場訪問対応
- スモールスタート：まず診断から、段階的な導入を支援
- 補助金活用：IT導入補助金など申請サポートも対応
