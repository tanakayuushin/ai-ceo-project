# 優先3業種 アプローチ戦略
**作成日：2026-05-06**  
**対象：飲食業・美容業・小売業（山口県内中小企業）**

---

## 全体の営業フロー

```
①発見（リスト作成）
　↓
②初回接触（Instagram DM / 直接訪問 / メール）
　↓
③無料相談（30分・オンラインor対面）
　↓
④無料診断レポート提出（AI活用ポイントをまとめた1枚）
　↓
⑤有料契約（月額サポートまたはスポット導入）
```

---

## 業種①：飲食業

### なぜ狙うか
- SNS集客の必要性を全員が感じている
- でも「毎日投稿するのがしんどい」という悩みが共通
- 即効性が見えやすく、費用対効果を説明しやすい

### リスト作成方法
1. Googleマップで「山口市 飲食店」「下関 居酒屋」などで検索
2. Instagramで地域タグ（#山口グルメ #下関グルメ）から発見
3. 食べログ・ホットペッパーで山口県の店舗一覧を取得

### アプローチチャネル（優先順）

**1位：Instagram DM（最も反応率が高い）**
```
件名なし・DM本文：

はじめまして。山口県でAI活用支援をしているEmport AIです。

○○さんのお店のInstagram、いつも拝見しています。
お料理の写真がとても素敵ですね。

一点ご提案があってご連絡しました。

「毎日の投稿文を考えるのが大変」という飲食店様が多く、
AIを使って投稿文を自動生成するサポートをしています。

30秒で1本分の投稿文が完成するので、
毎日更新が無理なく続けられるようになります。

もしご興味あれば、30分の無料相談をさせていただけませんか？
```

**2位：直接訪問（準備物：名刺＋A4一枚の案内）**
```
「こんにちは、山口でAI支援をしているEmport AIの者です。
飲食店様のInstagram投稿を、AIで自動化するサポートをしています。
今5分だけよろしいですか？」
```
→ 忙しそうなら「こちらだけ受け取ってください」と案内チラシを渡す

**3位：Googleビジネスプロフィール経由でメール**
- 連絡先メールが公開されている店舗に営業メールを送る

### 飲食業向け刺さるトーク

| 悩み | 提案 |
|------|------|
| 毎日投稿が続かない | 「AIが毎日の投稿文を30秒で生成します」 |
| Googleレビューの返信が面倒 | 「お客様のレビューに丁寧な返信をAIが自動作成」 |
| メニュー紹介の写真説明が苦手 | 「料理の写真を送るだけでキャッチコピーを生成」 |
| 予約の問い合わせ対応が大変 | 「LINEにAIチャットを導入して24時間自動返信」 |

## 業種②：美容業（美容室・ネイル・エステ）

### なぜ狙うか
- InstagramがメインのSNSであり、投稿の手間が大きい
- 個人・少人数経営が多く、SNS担当者がいない
- リピーター獲得のためのLINE配信に課題を感じている

### リスト作成方法
1. Instagramで「#山口美容室」「#山口ネイル」「#下関サロン」で検索
2. ホットペッパービューティーで山口県の店舗一覧を取得
3. Googleマップで「山口市 美容室」で検索

### アプローチチャネル（優先順）

**1位：Instagram DM**
```
はじめまして。山口県でAI活用支援をしているEmport AIです。

○○さんのサロンの作品、すごくおしゃれで参考になっています。

実は「スタイルの写真を撮ったあと投稿文を考えるのが大変」
という美容師さんが多くて、それをAIで解決するお手伝いをしています。

写真を1枚送るだけで、投稿文・ハッシュタグまでAIが自動生成します。

30分の無料相談で実際に体験してみませんか？
```

**2位：直接訪問**
- 来客が少ない時間帯（平日午前中）を狙って訪問
- オーナー・店長に直接話す

### 美容業向け刺さるトーク

| 悩み | 提案 |
|------|------|
| 投稿文・ハッシュタグが思い浮かばない | 「写真1枚でAIが投稿文を自動生成」 |
| LINEの配信文を考えるのが面倒 | 「月1回のLINE配信文をAIが作成」 |
| 新規集客に困っている | 「SEOに強いGoogleビジネスの説明文をAIで最適化」 |
| スタッフへの伝達・マニュアルが整備できていない | 「作業手順書をAIで自動生成」 |

## 業種③：小売業（実店舗・ネットショップ）

### なぜ狙うか
- 商品説明文・SNS投稿を大量に作る必要がある
- 少人数経営でライティングに時間を取られている
- AI活用で「量産」のメリットが即見える

### リスト作成方法
1. Googleマップで「山口市 雑貨店」「下関 セレクトショップ」で検索
2. BASE・minneなどのECサイトで山口県の出品者を探す
3. 地元の商店街を直接歩いて発見

### アプローチチャネル（優先順）

**1位：直接訪問**
- 小売店はSNSを見ていない場合が多い
- 実際に店舗に行って話すのが一番早い

**2位：Instagram / LINE公式アカウント DM**

```
はじめまして。山口県でAI活用支援をしているEmport AIです。

商品の紹介文やSNS投稿文を、AIを使って短時間で量産するサポートをしています。

「商品説明を書くのに時間がかかる」「SNS更新が追いつかない」
という悩みを持つお店様に喜ばれています。

30分の無料相談で実際にどれくらい効率化できるかお見せできます。
ご興味あればお気軽にご返信ください。
```

### 小売業向け刺さるトーク

| 悩み | 提案 |
|------|------|
| 商品説明文を書くのが時間かかる | 「商品名・特徴を入力するだけでAIが説明文を生成」 |
| SNS投稿が続かない | 「商品画像を送るだけで投稿文を自動生成」 |
| 広告文のバリエーションが作れない | 「AIで10パターンの広告文を瞬時に生成」 |

## 共通：無料相談のアジェンダ（30分）

```
【0〜5分】自己紹介・相手の事業確認
  「どんなお仕事をされているか教えてください」

【5〜15分】課題ヒアリング
  「今一番時間がかかっている業務はなんですか？」
  「SNSの更新はどれくらいの頻度でやっていますか？」
  「AIって使ったことありますか？」

【15〜25分】デモ実演（その場でAIを使ってみせる）
  → その店の投稿文をその場でAIに生成させる
  → 「これが30秒でできます」と見せる

【25〜30分】サービス説明・次のステップ提案
  「月額○万円でこれを毎月サポートできます」
  「まず1ヶ月だけ試してみませんか？」
```

**最重要：デモが一番の営業ツール。必ずその場でAIを動かして見せる。**

## 料金プランの提案例

| プラン | 内容 | 月額 |
|--------|------|------|
| ライトプラン | SNS投稿文の月10本生成・テンプレート提供 | 30,000円 |
| スタンダードプラン | SNS投稿文無制限＋LINE配信文作成＋月1回相談 | 50,000円 |
| フルサポートプラン | 上記＋AI導入設定・社内レクチャー込み | 100,000円 |

**補助金適用後の実質負担（デジタル化・AI導入補助金2026 補助率2/3の場合）**
- ライトプラン → 実質 約10,000円/月
- スタンダードプラン → 実質 約17,000円/月
