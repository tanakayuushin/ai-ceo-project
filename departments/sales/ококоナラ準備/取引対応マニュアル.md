# ококоナラ 取引対応マニュアル

**作成**: CEO アレン / 2026-04-30  
**根拠**: 実績出品者の分析・оокоナラ公式情報・高評価取引者のパターン調査

---

## ⚠️ 最重要ルール（必ず守ること）

| ルール | 理由 |
|--------|------|
| 購入から**48時間以内**に必ず返信する | 無返信で自動キャンセル→**全サービスが受付休止＋★1自動付与** |
| оокоナラ外（LINE・メール等）への誘導は絶対NG | 規約違反でアカウント停止になる |
| 修正回数・納品範囲は**事前に明確化**する | 「聞いてない」系トラブルの9割はここが原因 |

---

## 取引の全体フロー

```
購入通知
  ↓
① 購入直後メッセージ（24時間以内・目標1時間以内）
  ↓
② ヒアリング（情報収集）
  ↓
③ 作業・進捗報告
  ↓
④ 納品
  ↓
⑤ 確認・修正対応
  ↓
⑥ 取引完了＋レビュー依頼
  ↓
⑦ お礼メッセージ
```

## ① 購入直後メッセージ（最重要）

購入直後の一通が「この人に任せて良かった」という印象を決める。

### コピペ用テンプレート

```
〇〇様

この度はご購入いただき、誠にありがとうございます。
AI活用サポートを担当します、（名前）と申します。

丁寧・迅速に対応することをお約束します。どうぞよろしくお願いいたします。

さっそくですが、よりよいサポートのために、以下をお聞かせください。
（↓ サービスごとのヒアリング項目を貼り付け）

ご返信をお待ちしております。
お気軽にご質問いただければ幸いです。
```

### ポイント
- 名前で呼びかける（「〇〇様」）
- 自己紹介を一言入れる
- 安心感を伝える言葉を入れる
- ヒアリング項目をすぐに送る（二度手間を防ぐ）

---

## ② ヒアリング中の対応

### 返信が遅い購入者への対応
返信が2〜3日ない場合は一度だけ確認メッセージを送る。

```
〇〇様

先日お送りしたヒアリングシートのご確認はいかがでしょうか。
お急ぎでなければ問題ありませんが、ご不明点等ありましたらお気軽にどうぞ。
```

- **1週間以上返信なし**の場合はキャンセル申請も検討
- 強引に催促しない・感情的にならない

### 認識ずれを防ぐ確認

ヒアリング後、作業前に必ず内容を確認する。

```
ご回答ありがとうございます。
内容を確認しました。以下の認識で進めてよろしいでしょうか？

・〇〇について△△でサポートする
・納期は〇日以内
・修正は1回まで対応

よろしければ「了解です」とご返信いただけますと助かります。
```

---

## ③ 作業中の進捗報告（サービス2・3向け）

納期が3日以上かかる場合は中間報告を入れる。

```
〇〇様

現在レポートを作成しております。
〇月〇日（〇曜日）までにお届けできる予定です。
何かご不明な点があればお気軽にどうぞ。
```

→ 連絡がある出品者は評価が高い傾向がある（実績者分析より）

---

## ④ 納品メッセージ

```
〇〇様

お待たせいたしました。納品物をお届けします。

━━━━━━━━━━━
（↓ 成果物・アドバイス・レポートの内容）
━━━━━━━━━━━

ご不明な点や「ここをもう少し詳しく」といった点があれば、
取引完了前にお気軽にお申し付けください。
（修正は1回まで対応いたします）

〇〇様のお役に立てれば幸いです。
```

### 納品時のポイント
- 成果物を「---」等で見やすく区切る
- 修正回数の残りを明示する
- 「取引完了前に」と伝えておく（完了後の対応義務はない）

---

## ⑤ 修正対応

```
〇〇様

ご確認いただきありがとうございます。
ご指摘の「〇〇の部分」について修正いたします。
（修正回数：今回で1回目／修正対応はこれで終了となります）

修正版をお届けするまで少々お待ちください。
```

### 修正範囲の線引き（重要）
- **対応する**：最初のヒアリング内容の範囲内の修正
- **対応しない**：ヒアリング後に追加された新しい要求（スコープ外）
- スコープ外の要求には「追加オプション（有料）」として対応

```
〇〇様のご要望はよく理解できました。
ただ、今回のサービス範囲に含まれない内容となります。
別途ご相談いただければ対応可能ですので、お気軽にご連絡ください。
```

## ⑥ 取引完了＋レビュー依頼

```
〇〇様

この度はご利用いただき、誠にありがとうございました。
〇〇様のお役に立てたなら大変嬉しいです。

もしよろしければ、取引後のご評価・コメントをいただけますと
次の方への参考にもなり、大変励みになります。

また何かAIに関するご相談があれば、いつでもお声がけください。
引き続きどうぞよろしくお願いいたします。
```

### レビュー依頼のルール（実績者が実践）
- 「よろしければ」「もしよければ」と**お願いベース**にする
- 「★5をお願いします」は絶対NG（規約違反リスク）
- 感触が良かった相手には必ずお願いする
- ★4評価への「★5だと思った」というコメント返しも絶対NG

## トラブル別対応

| トラブル | 対応方法 |
|---------|---------|
| 返信がずっとこない | 2〜3日後に確認メッセージ1回 → 1週間でキャンセル検討 |
| 後出しで追加要求された | 「サービス範囲外」と丁寧に伝え、有料オプションを提案 |
| 低評価・クレームが来た | 感情的にならず事実確認。誠実に謝罪＋改善を提示 |
| 「外部で連絡したい」と言われた | 「оокоナラのルール上、外部連絡はできません」と断る |
| キャンセルを求められた | 着手前：応じる。着手後：部分対価を交渉 |

## 高評価★5をもらうための3原則（実績者分析まとめ）

1. **即レス**：返信が速いだけで信頼感が大幅に上がる。目標は1時間以内
2. **期待を超える**：「2,000円でここまでやってくれるの？」という驚きを与える。モニター期だからこそ全力投球
3. **人として誠実に**：わからないことを正直に言う、ミスはすぐ謝る、これだけで★5がとれる

- ❌ 48時間以上返信しない
- ❌ ококоナラ外での連絡に応じる
- ❌ 修正を無制限に受け入れる
- ❌ 感情的な返信をする
- ❌ ★5を強要・誘導する
- ❌ 取引完了後に成果物の無償追加対応をする
