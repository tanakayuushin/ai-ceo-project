# 台本②：セミナー本番（90分）

**用途**: 商工会議所会員企業向けAI活用入門セミナー  
**対象**: 中小企業の経営者・管理職（AIの知識はほぼゼロ前提）  
**目標**: 参加者にAIを身近に感じてもらい、AI業務診断への関心を引き出す

---

## 開始前の準備

- [ ] 名刺を人数分＋予備で用意
- [ ] 手元資料（サービス紹介・補助金一覧）を印刷
- [ ] 受付名簿（名前・会社名・連絡先）を準備
- [ ] 終了後の個別相談コーナーのスペース確保

## 第1部：なぜ今、中小企業にAIが必要か（15分）

```
「本日はお忙しい中お集まりいただきありがとうございます。
Emport AIのアレンと申します。

最初に少し聞かせてください。
『AIって自分の会社には関係ない』と思っている方、
正直に手を挙げていただけますか？

（間を置く）

ありがとうございます。
実は、そう思っていた経営者ほど、
後から『もっと早く知りたかった』とおっしゃいます。

今日お伝えしたいのはひとつだけです。
AIは大企業のものではありません。
むしろ、人手不足に悩む地方の中小企業にこそ、
今すぐ使える道具です。」
```

### 人手不足の現実（スライドまたは口頭）
```
「山口県の中小企業、特に製造業・サービス業では、
求人を出しても人が来ない、ベテランが辞めたら業務が回らない、
こういった声を本当によく聞きます。

AIは人を置き換えるものではなく、
今いる人の仕事を楽にする道具です。

たとえば——
・毎日1時間かかっていたメールの返信を10分にする
・請求書の入力作業を自動化する
・問い合わせの8割をAIが一次対応する

これは全部、山口県内の中小企業でも
今すぐできることです。」
```

## 第2部：AIで変えられる業務とは（25分）

### 身近な事例紹介
```
「具体的に見ていきましょう。

たとえば飲食店では——
予約の問い合わせ対応をAIチャットにしたことで、
深夜の予約受付が自動化されました。
スタッフは電話対応から解放されています。

製造業では——
検品作業でAIの画像認識を導入して、
ベテランが担っていた目視チェックを補助させています。
ベテランが休んでも品質が落ちない。

小売・サービス業では——
お客様からのよくある質問への返答を
AIが自動で作成し、スタッフが確認して送るだけ。
対応時間が半分以下になっています。」
```

### コストの現実
```
「次に気になるのがコストだと思います。

正直に言います。
最初から何百万もかける必要はありません。

今日からすぐ使えるAIツールは
月1万円以下のものもあります。

そして国の補助金を使えば、
導入費用の最大半額以上が補助される制度もあります。

IT導入補助金というものをご存じでしょうか——」
```

### 補助金の説明
```
「IT導入補助金は、中小企業がITツールを導入する際に
国が費用を補助してくれる制度です。

条件を満たせば、導入費用の50〜最大80%が戻ってきます。

私たちEmport AIは、この補助金申請のサポートも行っています。
手続きが面倒で使えていない企業様が多いので、
そこも一緒にやりましょうというのが私たちのスタンスです。」
```

## 第3部：どこから始めればいいか（20分）

```
「では、実際にどこから始めるか。

一番の失敗は『よくわからないまま高いツールを買う』ことです。

私たちが勧める第一歩は、AI業務診断です。

半日、皆さんの会社にお伺いして、
業務の流れをヒアリングさせていただきます。
そして『ここにAIが使える』というポイントをまとめた
レポートをお渡しします。

費用は5万円〜10万円です。
何百万も使う前に、まずここから始める。
それが一番リスクが低い進め方です。

今日ご参加の方には、
診断のご相談を特別に優先して対応させていただきます。
終了後に個別でお声がけください。」
```

## 質疑応答（30分）

### よくある質問と回答

**Q: うちの社員はITが苦手なんですが大丈夫ですか？**
```
「大丈夫です。今のAIツールはスマホと同じくらい
直感的に使えるものが多いです。
それに私たちは導入後の研修・サポートもやっています。
使いこなせるまで一緒にいます。」
```

**Q: セキュリティが心配です**
```
「大事なご指摘です。
社内の機密情報をそのままAIに入力するのは確かにリスクがあります。
どのデータをAIに使わせてよいか、どう設定するか、
そこも含めて診断の中でお話しします。
やみくもに使うのではなく、安全な使い方をお伝えします。」
```

**Q: 結局いくらかかりますか？**
```
「ツール・規模・業種によって全然違います。
だから診断が大切なんです。
まず診断で御社に合ったものを特定してから、
費用のご提案をします。
診断自体は5〜10万円。そこから先はゼロでもいいです、
合わないと判断すれば。」
```
