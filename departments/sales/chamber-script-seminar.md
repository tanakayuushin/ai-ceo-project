# 台本②：セミナー本番（90分）

**用途**: 商工会議所会員企業向けAI活用入門セミナー  
**対象**: 中小企業の経営者・管理職（AIの知識はほぼゼロ前提）  
**目標**: 参加者にAIを身近に感じてもらい、AI業務診断への関心を引き出す

---

## 開始前の準備

- [ ] 名刺を人数分＋予備で用意
- [ ] 手元資料（サービス紹介・補助金一覧）を印刷・人数分
- [ ] 受付名簿（名前・会社名・連絡先・業種）を準備
- [ ] ワークシート（自社に当てはめてみよう）を印刷・人数分
- [ ] スライドの動作確認・プロジェクター接続テスト
- [ ] 終了後の個別相談コーナーのスペース確保
- [ ] タイマーをセット（各パートの時間管理）

## オープニング（5分）

```
「皆さん、こんにちは。本日はお忙しい中お集まりいただき、
ありがとうございます。

Emport AIの代表、アレンと申します。
山口県の中小企業の皆さんのAI活用を支援するために、
この会社を立ち上げました。

今日は90分、皆さんと一緒に
『AIって結局うちの会社に使えるの？』
という問いに向き合っていきたいと思います。

まず最初に、少し手を挙げていただきたいのですが——

AIという言葉を聞いて
『うちには関係ない』と思っている方、いますか？

（間を置く、手が挙がったら）

正直に挙げてくださってありがとうございます。
その感覚は全く正しいと思います。
なぜなら、今まで世間のAIの説明は
大企業向けのものばかりだったからです。

今日はその誤解を全部解いて帰っていただきます。
どうぞよろしくお願いします。」
```

## 第1部：AIをめぐる3つの誤解（15分）

*※ここで「AIの誤解あるある」を先に潰すことで、後半が刺さりやすくなる*

### 誤解①「AIは大企業のもの」

```
「よくこう言われます。
『AIって、トヨタとか楽天みたいな大企業がやるものでしょ？』

実際には逆です。
大企業はすでにAIを使っています。
使い始めているのは中小企業です。今がその波です。

帝国データバンクの調査では、
従業員20名以下の小規模事業者でも
3社に1社がすでに何らかのAIツールを活用しています。

隣の競合がこっそり使い始めているかもしれません。」
```

### 誤解②「AIは何百万もかかる」

```
「費用についても誤解があります。

月額1万円以下で使えるAIツールがたくさんあります。
ChatGPTのビジネスプランは月3,000円です。
これだけで文書作成・メール返信・データ整理が
劇的に速くなります。

しかも国のIT導入補助金を使えば、
導入費用の最大75%が戻ってきます。

実質的な自己負担は思ったより全然小さい。」
```

### 誤解③「社員がITに弱いから無理」

```
「これが一番よく聞く声です。

でも今のAIは、スマホのアプリと同じ感覚で使えます。
インストールも設定も、昔のシステム導入とは別物です。

実際に私たちがサポートした企業では、
60代のパート従業員の方が2週間でAIツールを使いこなしました。

難しいのはツールではなく、
『何に使うか』を決めることです。
それを一緒に考えるのが私たちの仕事です。」
```

---

## 第2部：山口県の中小企業が直面している現実（10分）

```
「少し、皆さんの周りの話をさせてください。

山口県の有効求人倍率は全国平均を上回り続けています。
求人を出しても来ない、来ても続かない。
この状況は今後さらに厳しくなります。

2030年までに山口県の生産年齢人口は
さらに10%以上減少する見込みです。

つまり、今と同じ人数・同じやり方では
5年後に会社を回せなくなる可能性がある。

AIは『人を減らすため』の道具ではありません。
『今いる人で、今より多くのことをこなすため』の道具です。

1人が1.5人分働けるようになれば、
求人1人分の問題が解決します。
その手段としてのAIが、今最も現実的な選択肢です。」
```

## 第3部：業種別・AI活用の具体例（20分）

*※参加者の業種に合わせて強調するスライドを変える*

### 製造業

```
「製造業の皆さんに特に刺さる事例をお話しします。

【事例：部品メーカー・従業員15名】
課題：検品作業がベテラン2名に依存。その方たちが退職したら品質が保てない。
解決：AIカメラによる外観検査システムを導入。
結果：検品精度が向上し、ベテラン不在でも同水準の品質を維持。
     ベテランは高度な判断業務に集中できるようになった。
費用：導入費用150万円のうち、補助金で100万円を賄った。

【事例：食品製造・従業員30名】
課題：受発注のFAX処理に毎日2時間かかっていた。
解決：FAXをAIで自動読み取り・データ化するツールを導入。
結果：処理時間が2時間→20分に短縮。担当者を別業務へ配置転換できた。
費用：月額2万円のクラウドサービス。補助金で初期費用をカバー。」
```

### サービス業・小売業

```
「サービス業・小売の方には、こちらの事例が参考になります。

【事例：旅館・従業員20名】
課題：電話・メールの予約問い合わせ対応に毎日3〜4時間。
    フロントスタッフが本来の接客に集中できていなかった。
解決：AIチャットボットをホームページとLINEに設置。
結果：問い合わせの75%をAIが自動対応。
    スタッフは複雑な相談や接客に集中できるようになった。
     深夜の予約受付も自動化され、機会損失がなくなった。

【事例：小売店・従業員8名】
課題：週次の発注作業に時間がかかり、在庫の過不足が頻発。
解決：過去の販売データをAIで分析し、発注量を自動提案するツール導入。
結果：在庫ロスが30%削減。発注作業の時間が半分になった。」
```

### 建設業・その他

```
「建設業の方にも使える事例があります。

【事例：建設会社・従業員25名】
課題：工事の見積書作成に毎回3〜4時間かかっていた。
    過去の案件データが紙で管理されており参照しにくかった。
解決：過去の見積データをAIに学習させ、
    条件を入力すると見積の下書きを自動生成するシステムを構築。
結果：見積作成時間が1/3に短縮。提案数が増え、受注率も改善。

共通して言えることは——
AIは『特別な業種』のためのものではなく、
『書類・メール・データ処理・判断補助』が発生する業種なら
どこでも使えるということです。
ほぼすべての業種に当てはまります。」
```

*※ここで手元のワークシートを使う*

```
「少し手を動かしてみましょう。

（2〜3分待つ）

書けた方、いくつか教えていただけますか？

（参加者の回答を拾い、AIで解決できることをその場でコメント）

ありがとうございます。
皆さんが挙げてくださった作業のほとんどは、
実はAIが得意なことばかりです。」
```

## 第5部：補助金の活用方法（10分）

```
「ここで、費用の話を具体的にしておきます。

AI・ITツールの導入に使える補助金が今いくつかあります。

一番使いやすいのは『IT導入補助金』です。
中小企業・小規模事業者であれば対象になる可能性が高い。

補助率は最大75%。
つまり100万円のシステムを導入しても、
自己負担は25万円で済む計算です。

さらに、今日この会場の窓口である商工会議所さんが
申請のサポートをしてくださる
『小規模事業者持続化補助金』という制度もあります。

こちらは上限50万円で、補助率は2/3。
ホームページ制作や販促ツールにも使えます。

私たちEmport AIは、これらの補助金申請のサポートも行っています。
『補助金の手続きが面倒で踏み出せない』という方こそ、
ぜひご相談ください。一緒にやります。」
```

## 第6部：最初の一歩の踏み方（10分）

```
「最後に、実際にどうやって始めるかです。

私たちがお勧めする第一歩は『AI業務診断』です。

やることはシンプルです。
半日、皆さんの会社にお伺いします。
業務の流れを見せていただきながら、
『ここにAIが使える』というポイントを洗い出します。
1週間後に、レポートをお渡しします。

レポートには——
・AIで効率化できる業務のリスト
・どんなツールが合っているか
・費用と補助金の活用シミュレーション
・最初に手をつけるべき優先順位

これを全部まとめてお渡しします。

費用は5〜10万円。
その後どうするかは、皆さんが決めてください。
診断だけで終わってもかまいません。

大事なのは、やみくもに動く前に
自社の地図を持つことです。
その地図を作るのが、AI業務診断です。

今日ご参加の皆さんには、
診断の初回ご相談を優先的にお受けします。
終了後、名刺をお渡ししますのでお気軽に声をかけてください。」
```

## 質疑応答（30分）

### よくある質問と回答

**Q: AIって個人情報とか社内情報が漏れませんか？**
```
「非常に大切なご質問です。
ChatGPTなどの一般的なAIに、そのまま顧客情報や社内機密を
入力することは避けるべきです。これは事実です。

ただし、対策はあります。
・業務用の法人契約プランを使う（情報が学習に使われない設定ができる）
・社内データ専用のAI環境を構築する
・どの情報をAIに渡してよいか、ルールを決めて運用する

私たちの診断では、こうした安全な使い方の設計も含めてご提案します。
『使わない』より『正しく使う』が正解です。」
```

**Q: 導入したあとのサポートはありますか？**
```
「あります。導入して終わりではなく、
使いこなせるまで伴走します。

社内の担当者への研修、運用中のトラブル対応、
3ヶ月・6ヶ月後の効果検証まで含めてサポートします。
月額のサポート契約もご用意しています。」
```

**Q: うちの会社のデータが少なくてもAIは使えますか？**
```
「使えます。
AIはデータが大量になければ動かないと思われがちですが、
今主流の汎用AIツールはそんな条件がありません。

ChatGPTやCopilotのようなツールは、
データがなくてもすぐ業務に使えます。
診断でどのタイプのAIが合っているかをご提案します。」
```

**Q: 社員への説明・社内の反発が心配です**
```
「これは多くの経営者の方が悩む点です。

コツは『AIに仕事を奪われる』ではなく
『AIで楽になる』という伝え方をすることです。

実際、現場スタッフの方の多くは
面倒な繰り返し作業から解放されることを歓迎してくれます。

私たちはスタッフへの説明会・研修も支援していますので、
社内浸透の部分もお任せください。」
```

**Q: まず自分たちで試してみることはできますか？**
```
「もちろんです。
ChatGPTは無料プランから始められます。

まず経営者ご自身が1週間使ってみてください。
メールの下書き、議事録の整理、企画書の叩き台。
これだけでも体感が変わります。

ただ、『自社のどの業務に本格活用するか』の見極めは
診断を受けてからの方が確実です。
独学で遠回りするより、最初に地図を持つ方が早い。」
```
