## AI導入費用の「二重補助」で実質負担を最小化

```
┌─────────────────────────────────────────────────────┐
│         AI導入 総費用：500万円 の場合               │
│                                                     │
│  ① デジタル化・AI導入補助金2026（国）              │
│     補助上限：最大 450万円                          │
│     補助率：1/2〜2/3                                │
│     → 500万円 × 2/3 ＝ 333万円 補助               │
│                                                     │
│  ② やまぐち産業振興財団 DX補助金（県）             │
│     補助上限：最大 50万円                           │
│     補助率：1/2                                     │
│     → 残額167万円 × 1/2 ＝ 50万円 補助（上限）    │
│                                                     │
│  ─────────────────────────────                     │
│  合計補助額：333万円 + 50万円 ＝ 383万円           │
│  実質自己負担：500万円 − 383万円 ＝ 117万円        │
└─────────────────────────────────────────────────────┘
```

**500万円のAI導入が、実質 約117万円 の負担で実現できます。**

## よくある質問

**Q: 2つの補助金を同時に申請できますか？**
A: 条件が揃えば国と県の補助金を併用活用できます。Emport AIが申請戦略を設計します。

**Q: 採択されなかった場合は？**
A: 補助金なしでの導入も検討しますが、採択率向上のためのサポートを万全に行います。

**Q: 申請の手間はどのくらいですか？**
A: Emport AIが書類作成・申請手続きをサポートします。お客様の負担は最小限です。
