## 進捗ステージ別 自己紹介文

メール作成時は `見込み客リスト.md` の契約社数を確認し、該当ステージの文面を使う。

### Stage 1 ── 契約0社（現在）
```
山口県を拠点に、中小企業向けのAI活用支援での起業を目指している
Emport AI（個人事業主）の田中悠清と申します。
```

### Stage 2 ── パイロット1〜5社
```
山口県を拠点に、中小企業向けのAI活用支援を始めました
Emport AI（個人事業主）の田中悠清と申します。
```

### Stage 3 ── 契約5〜20社（実績あり）
```
山口県を拠点に、中小企業向けのAI活用支援を行っております
Emport AI（個人事業主）の田中悠清と申します。
山口県内{{契約社数}}社様のAI導入をご支援した実績がございます。
```

### Stage 4 ── 契約20社以上（法人化後）
```
山口県を拠点に、中小企業向けのAI活用支援を行っております
Emport AI 代表の田中悠清と申します。

補助金活用から導入・運用まで一貫してサポートしています。
```

## 件名パターン

```
【AI活用のご提案】{{会社名}}様の{{課題・業務}}を効率化できるかもしれません
```

例：
- 【AI活用のご提案】山口製作所様の検品業務を効率化できるかもしれません
- 【AI活用のご提案】○○旅館様の問い合わせ対応を自動化できるかもしれません

## カスタマイズポイント

AIが以下を会社ごとに差し替えて生成する:

| プレースホルダー | 差し替え内容 |
|----------------|-------------|
| `{{会社名}}` | 実際の会社名 |
| `{{役職}} {{担当者名}}` | 代表取締役 / ご担当者 など |
| `{{業種・事業}}` | 具体的な事業内容 |
| `{{具体的な課題や状況への言及}}` | 採用状況・事業規模・ニュースなど |
| `{{業種に応じた具体的なAI活用事例}}` | Stage3以上のみ使用 |
| `{{契約社数}}` | 見込み客リストの「✅ 契約」件数 |

---

## ステージ判定ルール（アレン用）

メール生成時に `見込み客リスト.md` を読み込み、ステータスが「✅ 契約」の社数を数える。

| 契約社数 | 使用ステージ |
|---------|------------|
| 0社 | Stage 1 |
| 1〜4社 | Stage 2 |
| 5〜19社 | Stage 3 |
| 20社以上 | Stage 4 |
