## 📝 撮影・制作メモ

### 必要素材
- [ ] オーナー役のモデル（または本人登場）
- [ ] スマートフォン（Android）画面録画
- [ ] BGM: 落ち着いたビジネス系インスト（フリー素材）
- [ ] ロゴデータ（Emport AI）

### 編集ポイント
- フォント: Bold、白テキスト、ネイビー背景
- カラー: #0A2342（ネイビー）、#F0A500（ゴールド）
- トランジション: シンプルなフェードイン
- テンポ: 情報量多めだが読めるスピードで

### 配信戦略
1. **YouTube** → フル2分30秒版

3. **X（Twitter）** → テキスト+画像でチャット例を紹介
4. **商談資料** → PDF版スクリーンショット集
