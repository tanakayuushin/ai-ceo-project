# SNSフォロワー成長戦略（Emport AI版）

**作成**: CEO アレン / 2026-05-19  
**対象**: X（Twitter）・LinkedIn  
**目的**: フォロワーを伸ばし、中小企業オーナーへのリーチを最大化する

---

## 1. 現状診断（2026-05-19時点）

| 指標 | 現状 | 問題 |
|------|------|------|
| 最大インプレッション | 31 | 桁が2つ足りない |
| いいね・RT | 0 | エンゲージメント完全ゼロ |
| 投稿スタイル | 一方向発信のみ | 返信・交流が皆無 |
| 外部リンク | 投稿本文に埋め込み | アルゴリズムがスコアを下げる |
| 投稿テーマ | 複数混在 | AIが専門性を認識できていない |

**根本原因**: 「発信だけして交流しない」状態。Xのアルゴリズムは会話（リプライ往復）を最重視するため、どれだけ良い投稿をしても届かない。

## 2. プラットフォーム戦略

### X（メイン）
- **なぜ優先**: 中小企業オーナー・経営者層が情報収集に使用。バズれば地域外にも拡散
- **弱点**: 初期フォロワーゼロからの立ち上げが難しい。交流なしでは絶対に伸びない
- **戦略**: 70/30 リプライ戦略 + 専門テーマ特化

### LinkedIn（サブ・中期）
- **なぜ注目**: 日本国内ユーザー500万人突破（2026年）。BtoBの意思決定者層が集中
- **優位性**: 「部長以上」「経営者」に直接リーチ可能。企業より個人アカウントの方がエンゲージメント高
- **戦略**: 週2〜3投稿から開始。X と同じコンテンツを転用可能

## 3. Xアルゴリズム 2026年版 完全攻略

### エンゲージメントの重み付け（公式ソースコードより）

| アクション | アルゴリズムスコア |
|---------|--------------|
| **リプライ往復（会話）** | **いいねの75〜27倍** |
| ブックマーク | 高 |
| リツイート | 中 |
| いいね | 低 |
| 外部リンクあり | マイナス |

### 投稿後30分が全て
- 投稿直後30分のエンゲージメント速度でリーチが決まる
- 投稿したらすぐ他の投稿にリプライ → 自分の投稿も表示されやすくなる
- フォロワーに「見て！」と促せる仕組みを作る（返報性を活用）

### やってはいけないこと
- 投稿本文にURL（外部リンク）を貼る → 代わりに**最初のリプライ（自己返信）**にURLを貼る
- 毎日テーマをバラバラに投稿 → AIが「このアカウントは何屋か」を認識できない
- リプライせず、いいねだけ返す → 会話にならないのでスコアゼロ

### アルゴリズムが好む投稿形式
1. **スレッド（連続ツイート）** → 単投稿の3倍エンゲージメント
2. **数字＋箇条書き** → 具体性があり保存・引用されやすい
3. **問いかけ形式** → 返信を誘発する
4. **画像・図解あり** → テキストのみより高スコア

### X Premium の効果
- フォロワーへの表示：**4倍ブースト**
- 非フォロワーへの表示：**2倍ブースト**
- フォロワー1,000人未満の段階では特に費用対効果が高い（月1,000円程度）

## 4. 70/30 リプライ戦略（最重要）

### 基本構造
```
時間配分:
  70% → 他アカウントへのリプライ・交流
  30% → 自分のオリジナル投稿
```

### なぜリプライが最強か
- 大アカウントにリプライすると、その大アカウントのフォロワー数万人に自分のコメントが見える
- 内容が良ければ「このアカウント面白い」→プロフィールを確認→フォロー、の流れが生まれる
- 実績: ある発信者が500フォロワーから6ヶ月で12,000人に成長した手法

### リプライの質と量

| 状況 | 1日の目標リプライ数 |
|------|----------------|
| フォロワー〜100人 | 30〜50件 |
| フォロワー100〜500人 | 20〜30件 |
| フォロワー500〜1,000人 | 15〜20件 |

### リプライの「質」を上げる公式
```
❌ 悪い: 「参考になりました！」「そうですね」
✅ 良い: 「感想 + 自分の経験/知識 + 質問」

例:
「AIの業務効率化、まさにその通りです。
 先月山口の飲食店でシフト管理をAI化したら
 月15時間の削減ができました。
 御社ではどの業務から始めましたか？」
```

## 5. コンテンツ戦略（Emport AI特化版）

### 投稿テーマを1本に絞る
```
「山口の中小企業がAIで業務改善する」専門アカウント
```
AIが「このアカウント＝山口中小企業AI」と認識するまでテーマをブラさない。

### 黄金フォーマット（バズ率高い順）

#### 1位: 事例紹介型スレッド（最強）
```
【実例】山口の〇〇業（スタッフ〇名）でAIを導入した話

✅ 課題: 〇〇に毎月〇時間かかっていた
✅ 導入: 〇〇ツールを〇〇方法で使用
✅ 結果: 〇時間 → 〇分（削減率〇%）

詳細は↓のスレッドで👇

---（スレッド続き）---
① 具体的な手順
② コスト感
③ 注意点
④ 今後の展望
```

#### 2位: 数字ありTips型
```
ChatGPT活用で変わった3つのこと

① メール作成: 30分→3分
② 会議議事録: 1時間→5分
③ 求人票作成: 半日→15分

プロンプトのコツは「役割+状況+出力形式」を指定すること。

試したことない方は明日から試してみてください。
```

#### 3位: 問いかけ型（エンゲージメント収集）
```
中小企業の経営者さんに質問です。

AIツール、今どのくらい使っていますか？

→ 毎日使ってる
→ たまに使う
→ 試したけど続かない
→ まだ使ったことない

コメントで教えてください。
今後の発信の参考にします。
```

### 外部リンクの正しい貼り方
```
投稿本文: URL不要（テキストと価値だけ）
    ↓
自己リプライ（スレッドの最後）: 
「詳しくはこちら → [URL]」
「無料相談はプロフのリンクから」
```

## 6. 具体的な1日のアクション（実行プロトコル）

### 朝の投稿セッション（30分）

```
07:00 - 投稿
  └ 事例紹介 or Tips型（スレッド形式）
     └ 本文にURLなし
     └ 必要なら最初のリプライにURL

07:10〜07:30 - 投稿直後の交流（重要）
  └ ターゲットアカウント10件にリプライ
  └ 「感想 + 知識/経験 + 質問」の質リプライ
  └ 15分以内の返信で3〜5倍リーチアップ
```

### 昼の交流セッション（15分）

```
12:00 - オプション投稿（問いかけ or 補足情報）
12:05〜12:20 - リプライ10件
  └ 朝のリプライに返信があれば必ず返す
  └ 新しいターゲットアカウントに5〜10件
```

### 夜の交流セッション（15分）

```
18:00〜18:15 - リプライ10件
  └ 今日の投稿を振り返る
  └ いいね・RTしてくれた人にお礼リプライ
  └ ターゲットアカウントの夜の投稿にリプライ
```

**1日の合計**: リプライ30件 + 投稿1〜2本
