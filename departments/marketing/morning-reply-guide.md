## 概要

X（Twitter）でのフォロワー獲得の核心は **「返信 7 : 自分の投稿 3」** の比率。  
このスクリプトは対象アカウントの最新ツイートを取得し、Claude Haiku が返信案を3パターン提示する。  
あなたは選ぶだけ（または軽く編集して）投稿できる。

---

## 手順

### 1. ターミナルを開く

PowerShell または VSCode のターミナルで ai-ceo-project フォルダに移動。

```powershell
cd "C:\Users\tsube\OneDrive\デスクトップ\ai-ceo-project"
```

### 2. スクリプトを実行

```powershell
python tools/social-media/morning_reply.py
```

### 3. インタラクティブな操作

スクリプトがアカウント順に最新ツイートと返信案を表示する。

```
  ツイート (2026-05-20 08:30):
  AIを活用した業務改善は…（ツイート本文）
  https://x.com/shota7180/status/xxxxx

  [1] (87/280)
  おっしゃる通りです。弊社でも…（フォーマル案）

  [2] (72/280)
  それ、すごくわかります！…（カジュアル案）

  [3] (65/280)
  興味深い視点ですね…（質問特化案）

  操作: [1/2/3]=投稿  [e]=編集  [s]=スキップ  [q]=終了
  > 
```

| 入力 | 動作 |
|------|------|
| `1` / `2` / `3` | そのまま投稿 |
| `e` | 自由に文章を入力して投稿 |
| `s` または Enter | このツイートをスキップ |
| `q` | 終了 |

## 返信の黄金フォーマット

```
感想（1行） + 経験・知識（1〜2行） + 質問（1行）
```

**例（中小企業DX投稿への返信）:**
```
共感します。地方の中小企業だと IT 担当が兼務のケースが多く、
最初の一歩がハードル高めです。
山口では製造業の単純記録系タスクから始めると定着率が高い印象があります。
御社では最初にどの業種・業務から始めましたか？
```

**NG パターン:**
- サービス・URL の宣伝
- 同じ文章を複数アカウントに貼り付け
- 「いいね・RTして」などエンゲベイト

## アカウントの追加方法

`tools/social-media/target_accounts.yaml` を編集する:

```yaml
tier_b:
  - username: 新しいアカウント名
    name: 表示名
    followers: 10000
    theme: テーマ概要
    reply_angle: 返信のアングル
```

## トラブルシューティング

| 症状 | 原因 | 対処 |
|------|------|------|
| `X_CONSUMER_KEY` エラー | .env が未設定 | .env に X API キーを追加 |
| `ANTHROPIC_API_KEY` エラー | APIキー未設定 | .env に ANTHROPIC_API_KEY を追加 |
| 403 Forbidden | ツイートが長すぎ | 130字以内に編集 |
| ツイートが表示されない | 直近24h に投稿なし | スキップ（正常動作） |
| Rate limit | API制限 | 15分待ってから再実行 |

## 補足: なぜ完全自動化しないのか

X の利用規約では自動的なリプライは禁止されている。  
アカウント凍結のリスクがあるため、**最終判断はアレン（人間）が行う半自動ワークフロー**として運用する。  
スクリプトは「草案を提示する秘書」の役割。
