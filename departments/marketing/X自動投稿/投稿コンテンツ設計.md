# X 投稿コンテンツ設計書
## 〜フォロワーにとって有用な情報を軸にした発信戦略〜

**作成**: CEO アレン / 2026-05-01  
**基本方針**: 80%が「フォロワーの役に立つ情報」、20%が「サービス告知」

---

## コンテンツの大原則

> **「自分の宣伝より、相手の問題解決を優先する」**

フォロワーが増え、信頼が生まれ、その結果として仕事の依頼が来る。
この順番を守ること。宣伝から始めると誰にも届かない。

---

## 投稿カテゴリ（比率設計）

| カテゴリ | 比率 | 目的 |
|---------|------|------|
| A. 即使えるAI活用Tips | 40% | フォロワーが「これ試してみよう」と思える実用知識 |
| B. 業種別の業務改善ネタ | 20% | 飲食・美容・整骨院などターゲット層に刺さる情報 |
| C. 小規模ビジネス経営知識 | 15% | 集客・採用・コスト削減など経営に役立つ視点 |
| D. 共感・気づき系 | 5% | 「わかる」「そうだった」と感じてもらえるひと言 |
| E. サービス告知 | 15% | okokoナラのサービス紹介・実績報告 |
| F. 質問・アンケート | 5% | エンゲージメント向上・フォロワーの声を拾う |

## カテゴリ別 投稿テンプレート集

### A. 即使えるAI活用Tips（週3〜4投稿）

**フォーマット：**
```
【ChatGPT活用Tips #〇〇】

（具体的にできること or よくある失敗）

→ このプロンプトを試してください：

「〇〇について、〇〇の立場から〇〇の形式で教えてください」

（結果の変化や効果の一言）

#ChatGPT活用 #AI業務効率化 #生成AI
```

**ネタ例：**
- メール返信を5秒で下書き
- 議事録の要点を30秒でまとめる
- クレーム対応文章をワンクリックで
- 企画書のたたき台を3分で作る
- 求人票を魅力的に書き直す


## 投稿の品質チェックリスト

投稿する前に確認する：

- [ ] 「このツイートを見た人は、何かメリットを得られるか？」
- [ ] 宣伝的な言葉は最小限になっているか？
- [ ] 140字（1ツイート）に収まっているか、または意図的に複数ツイートに分けているか
- [ ] ハッシュタグは2〜3個以内か（多すぎるとスパムに見える）
- [ ] 数字・具体例が入っているか（抽象的な言葉だけだと読まれない）
- [ ] 読んだ人が「保存したい」「リツイートしたい」と思えるか

## NG投稿パターン

| NG | 理由 |
|----|------|
| 「サービスを始めました！よろしくお願いします！」だけ | 情報ゼロ・誰の役にも立たない |
| 毎日サービス告知 | フォロー解除される |
| ハッシュタグを10個以上つける | スパム判定・見苦しい |
| 長文を1ツイートに詰め込む | 読まれない |
| 他社の批判・ネガティブな発言 | 信頼を損なう |
| AIが書いたまま投稿（明らかにAI文体） | 人間味がなく距離ができる |
