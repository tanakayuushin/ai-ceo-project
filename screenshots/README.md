# Emport AI スクリーンショット管理

**用途:** デモ・面談・資料・SNS投稿などに使うスクショをここに集める  
**ルール:** 撮ったらすぐ対応フォルダに入れる

---

## フォルダ構成

### 📁 01-website
**公式Webサイトのスクショ**
- emport-ai.vercel.app のトップページ
- サービス紹介セクション
- 問い合わせ・CTA部分
- PC版 / スマホ版それぞれ

### 📁 02-booking-system
**予約ページ・Jicooのスクショ**
- booking.html（予約ボタンページ）
- Jicooの予約カレンダー画面
- 予約完了画面
- Jicoo管理画面（設定・統計）

### 📁 03-ai-systems
**AIシステム稼働画面のスクショ**
- 日次ブリーフィング（ターミナル or ファイル表示）
- Claude Codeの動作画面
- インテリジェンスレポートの出力画面
- 自動タスク実行の様子

### 📁 04-x-twitter
**X（旧Twitter）関連のスクショ**
- X公式アカウント（@AI_chuusyou）のプロフィール画面
- 投稿一覧・エンゲージメント
- 個別ツイートのインサイト
- フォロワー推移グラフ

### 📁 05-mobile-app
**Emport AIモバイルアプリのスクショ**
- iOS / Android 画面
- 業種別AIアシスタント画面
- チャット画面
- ログイン・トップ画面

### 📁 06-tools-webapp
**業務支援Webアプリ（Flask）のスクショ**
- ログイン画面
- 4タブのメイン画面
- 各ツールの動作画面
- Railway管理画面（稼働状態）

### 📁 07-meeting-demos
**面談・デモ用スクショ（山口大学など）**
- 山口大学面談で見せる画面一式
- プレゼン用に切り出したスクショ
- 「before/after」比較用

### 📁 08-competitors
**競合・市場調査のスクショ**
- 競合サービスの画面
- 市場データ・グラフ
- ニュース記事のキャプチャ

### 📁 09-analytics-reports
**数値・分析レポートのスクショ**
- X投稿のアナリティクス
- Webサイトのアクセス解析
- Railway のサーバー稼働状況
- freee / 財務データ
- Looker Studio ダッシュボード

### 📁 10-sales-outreach
**営業・問い合わせフォーム送信のスクショ**
- 見込み客への問い合わせ送信確認画面
- 営業活動の証跡

### 📁 11-email-gmail
**Gmail・メール関連のスクショ**
- Gmail設定・ラベル・フィルター
- 送受信メールの確認画面
- メール自動化の設定

### 📁 12-crm-hubspot
**HubSpot CRM関連のスクショ**
- HubSpot設定・ダッシュボード
- CRM連携画面
- 顧客管理設定
