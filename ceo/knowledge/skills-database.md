```

       ↓

       ↓
⑤ 交付申請（IT導入支援事業者がポータルで代行入力）
       ↓
⑥ 交付決定（審査：約1〜2ヶ月）
       ↓
⑦ ITツール導入・支払い
       ↓
⑧ 事業実績報告
       ↓
⑨ 補助金振込
```

### 採択されやすい申請のポイント

- **AI活用を明示する**: 導入ツールにAI機能が含まれることを具体的に記述
- **業務改善効果を数値化**: 「作業時間を週◯時間削減」「月◯万円のコスト削減」を具体的に
- **賃上げ要件**: 補助率4/5を狙うなら賃上げ計画を事業計画に含める
- **小規模事業者優遇**: 従業員20名以下は補助率が高い

### やまぐち産業振興財団 DXツール導入型補助金

- 上限: 50万円 / 補助率: 1/2
- 随時受付（先着約50件/期）
- 国補助金との併用可（二重申請NG・同一経費への重複補助はNG）
- 申請要件: 山口県内に主たる事業所がある中小企業

## 4. 自動化に使えるAPI実践ガイド

### freee API（帳簿自動化）

```python
import requests

# freee APIトークン（OAuth2.0で取得）
headers = {"Authorization": f"Bearer {ACCESS_TOKEN}"}

# 取引（収入）を登録
def create_deal(amount, description, date):
    url = "https://api.freee.co.jp/api/1/deals"
    payload = {
        "company_id": COMPANY_ID,
        "issue_date": date,  # "2026-05-10"
        "type": "income",
        "details": [{
            "account_item_id": SALES_ACCOUNT_ID,
            "amount": amount,
            "description": description
        }]
    }
    res = requests.post(url, json=payload, headers=headers)
    return res.json()
```

## 5. 日本中小企業経営者へのアプローチ術

- **最初の15秒が全て**: 「◯◯を自動化して月◯時間削減できます」と具体的数字から入る
- **補助金は最強の入口**: 「実質負担ゼロ」は決裁を早める最大の動機付け
- **競合他社の事例を使う**: 「同業の◯◯社さんが導入して〜」という話が刺さる
- **IT用語を使わない**: 「AI」より「自動化」「省力化」「楽になる」の方が伝わる
- **小さく始める提案**: 「まず1つの業務だけ試しましょう」から入るのが鉄則
- **決裁者に直接**: 担当者ではなく社長・経営者に直接話す機会を作る
