# LINE リマインダーボット セットアップ手順

## 全体の流れ（約20分）

```
① LINE Developers でボット作成（5分）
② Google スプレッドシート作成（1分）
③ Google Apps Script にコード貼り付け（5分）
④ スクリプトプロパティ設定（3分）
⑤ ウェブアプリとしてデプロイ（3分）
⑥ LINE WebhookにURLを登録（2分）
⑦ トリガー設定（2分）
```

---

## ① LINE Developers でボット作成

1. https://developers.line.biz にアクセス（LINEアカウントでログイン）
2. 「コンソール」→「新規プロバイダー作成」→ 名前を入力（例：「個人ボット」）
3. 「Messaging API チャンネルを作成」をクリック
4. 必須項目を入力：
   - チャンネル名：「リマインダーくん」など
   - 業種：「個人」
5. 作成後、「Messaging API設定」タブを開く
6. 「チャンネルアクセストークン（長期）」の「発行」をクリック → **コピーして保存**

---

## ② Google スプレッドシート作成

1. https://sheets.google.com で新しいスプレッドシートを作成
2. 名前を「LINEリマインダー」などに変更
3. URLから **スプレッドシートID** をコピーして保存
   ```
   https://docs.google.com/spreadsheets/d/【ここがID】/edit
   ```

---

## ③ Google Apps Script にコード貼り付け

1. スプレッドシートのメニュー「拡張機能」→「Apps Script」を開く
2. デフォルトのコードをすべて削除
3. `Code.gs` の内容をすべてコピーして貼り付け
4. 「保存」（Ctrl+S）

---

## ④ スクリプトプロパティ設定

1. Apps Script の左メニュー「プロジェクトの設定」（歯車アイコン）
2. 「スクリプト プロパティ」→「プロパティを追加」で以下3つを登録：

| プロパティ名 | 値 |
|------------|---|
| `LINE_TOKEN` | ①でコピーしたチャンネルアクセストークン |
| `SHEET_ID` | ②でコピーしたスプレッドシートID |
| `LINE_GROUP_ID` | （空欄でOK・ボットをグループに追加後に自動保存される） |

---

## ⑤ ウェブアプリとしてデプロイ

1. Apps Script 右上「デプロイ」→「新しいデプロイ」
2. 種類：「ウェブアプリ」
3. 設定：
   - 実行ユーザー：**自分**
   - アクセスできるユーザー：**全員**
4. 「デプロイ」→ Googleアカウントを承認
5. 表示された **ウェブアプリのURL** をコピーして保存

---

## ⑥ LINE WebhookにURLを登録

1. LINE Developers の「Messaging API設定」タブ
2. 「Webhook URL」に⑤のURLを貼り付け
3. 「Webhookの利用」を **ON** に
4. 「検証」ボタンで「成功」と出ればOK
5. 「応答メッセージ」は **OFF** に（自動応答を無効化）
6. 「あいさつメッセージ」も **OFF** に

---

## ⑦ 定期実行トリガー設定（リマインド自動送信用）

1. Apps Script 左メニュー「トリガー」（時計アイコン）
2. 「トリガーを追加」
3. 設定：
   - 実行する関数：`sendReminders`
   - イベントのソース：「時間主導型」
   - 時間ベースのトリガーのタイプ：「1時間おきのタイマー」
4. 「保存」

---

## ⑧ ボットをLINEグループに追加

1. LINE Developers「Messaging API設定」→ QRコードを表示
2. スマホのLINEでQRコードを読み込んで友達追加
3. リマインドしたいグループにボットを招待
4. グループで `/help` と送信 → ヘルプが返ってきたら完了！

---

## 使い方

```
/add 定例会 2026/05/10 19:00        → 1回限りのイベント登録
/add 練習 毎週木曜 18:30            → 毎週繰り返し
/add 会費集め 毎月1日 12:00         → 毎月繰り返し

/list                               → 今後の予定一覧

/del 1                              → 番号1のイベントを削除

/help                               → ヘルプ表示
```

## リマインドのタイミング
- **前日夜20時**：「明日の予定があります！」
- **当日朝8時**：「今日の予定！遅刻しないように〜！」

---

## TimeTree 連携：予定を自動でスプレッドシートに取り込む

TimeTree の予定を1時間おきに自動取得し、スプレッドシートに登録→LINEリマインドします。

### ① TimeTree アクセストークンを取得

1. https://developers.timetreeapp.com にアクセス（TimeTree アカウントでログイン）
2. 「Create Application」→ アプリ名・説明を入力して作成
3. 作成後、**OAuth Token** または **Personal Access Token** を発行してコピー

### ② スクリプトプロパティに追加

Apps Script「プロジェクトの設定」→「スクリプト プロパティ」に以下を追加：

| プロパティ名 | 値 |
|------------|---|
| `TIMETREE_TOKEN` | ①でコピーしたトークン |
| `TIMETREE_CALENDAR_ID` | （次の手順で確認） |

### ③ カレンダーID を確認

1. `TIMETREE_TOKEN` だけ登録した状態でスプレッドシートを開く
2. メニュー「リマインダー」→「TimeTree カレンダーID を確認」を実行
3. 表示されたダイアログのIDをコピーして `TIMETREE_CALENDAR_ID` に登録

### ④ 自動同期トリガーを設定（1回だけ）

メニュー「リマインダー」→「TimeTree 自動同期トリガーを設定」を実行

→ 以降は **1時間おきに自動で TimeTree の予定を取得 → シートに追記 → LINEリマインド**

### 動作イメージ

```
TimeTree（スマホで予定追加）
       ↓ 最大1時間後に自動取得
スプレッドシート（eventsシートに自動追記）
       ↓ 1時間おきに sendReminders() が判定
LINEグループ（前日20時・当日8時にリマインド通知）
```

### 注意事項

- TimeTree API は**今後7日分**の予定を取得します。1週間以上先の予定は7日以内に入ったタイミングで自動取得されます
- すでに同じ「イベント名＋日付」がシートにある場合は重複追加しません
- 終日イベントは 09:00 として登録されます
- 手動同期はメニュー「TimeTree から同期」からいつでも実行できます

## スマホ対応：Web アプリ URL からアクセスする

> スマホの Google Sheets アプリはカスタムメニュー・サイドバー非対応のため、Web アプリ URL を使います。

### 手順

1. **Web アプリとして再デプロイ**（既にデプロイ済みの場合は URL を確認するだけでOK）
   - Apps Script 右上「デプロイ」→「デプロイを管理」
   - 表示されている **ウェブアプリの URL** をコピー

2. **スプレッドシートにリンクを貼る**
   - スプレッドシートの A1 セルなど目立つ場所を選択
   - 以下の数式を入力（URL は自分のものに差し替え）：
     ```
     =HYPERLINK("https://script.google.com/macros/s/【デプロイID】/exec","▶ イベントを追加（スマホ用）")
     ```
   - スマホで Sheets を開いてこのセルをタップ → ブラウザでフォームが開く

3. **ブックマークに追加（任意）**
   - ブラウザでフォームを開いた後、ホーム画面にショートカットを追加すると次回からワンタップでアクセスできる

### 動作の仕組み

| 環境 | 使い方 |
|------|--------|
| PC（ブラウザ） | スプレッドシートのメニュー「リマインダー」→「イベントを追加」 |
| スマホ（Sheets アプリ） | スプレッドシート内の HYPERLINK セルをタップ → ブラウザでフォームが開く |
